# Usage of Multiple Widget & Events

#### Using Web Browser(1)
1. 웹브라우저 사용하기
  - 웹브라우저는 롤리팝에서 크롬 브라우저 내장
  - HTML5 표준 태그들을 이용한 기능이 지속적으로 추가될 것임
  - 웹브라우저를 애플리케이션 안에 넣고 싶은 경우에는 웹뷰를 사용하면 되는데, XML 레이아웃에서는 WebView 태그로 정의함
